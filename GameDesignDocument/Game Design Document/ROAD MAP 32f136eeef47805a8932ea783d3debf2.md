# ROAD MAP

- Setup project Unity 2022.3 LTS
- Import path asset, test player chạy theo path
- Viết PlayerController: forward + lane switch (3 lane)
- Viết BotController: chạy theo path, tính khoảng cách Z
- Thêm Item đơn giản: Code Commit (+điểm)
- Thêm Obstacle đơn giản: chạm → slow
- Win/Lose trigger box cuối path
